Conectar con Google Sheets

1. Ir a console.cloud.google.com → nuevo proyecto
2. APIs y servicios → Sheets API → Habilitar
3. Credenciales → Crear → ID de cliente OAuth → Aplicacion web
4. Origen JS autorizado: abriendo...
5. URI de redireccionamiento: abriendo...
